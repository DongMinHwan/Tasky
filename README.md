# 📌 TodoList iOS App (with RxSwift + MVVM + Node.js + MongoDB)

연습용으로 제작한 간단한 TodoList 앱입니다.  
RxSwift와 MVVM 아키텍처 패턴을 적용하여 구조화하였고,  
Node.js + MongoDB 기반의 커스텀 서버를 직접 구현하여 RESTful API를 통해 데이터를 주고받도록 구성했습니다.

## 🛠️ 기술 스택

### iOS (Client)
- Swift + UIKit
- RxSwift
- MVVM 아키텍처
- Moya

### Backend (Server)
- Node.js + Express
- MongoDB (로컬 or Atlas)
- RESTful API (GET / POST / DELETE / PUT)

## 📱 주요 기능
- 할 일 목록 조회
- 할 일 추가
- 할 일 삭제
